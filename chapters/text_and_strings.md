## Manipulation des chaînes

Voici quelques exemples de кодов (code) en Python pour manipuler les chaînes de caractères:

### Exemple 1: Concaténation de chaînes

```python
str1 = "Bonjour"
str2 = "tout le monde"
result = str1 + " " + str2
print(result)  # Affiche: Bonjour tout le monde
```

### Exemple 2: Longueur d'une chaîne

```python
str1 = "Salut"
print(len(str1))  # Affiche: 5
```

### Exemple 3: Conversion à la majuscule

```python
str1 = "bonjour"
print(str1.upper())  # Affiche: BONJOUR
```

### Exemple 4: Remplacer une sous-chaîne

```python
str1 = "Bonjour tout le monde"
result = str1.replace("monde", "Python")
print(result)  # Affiche: Bonjour tout le Python
```

### Exemple 5: Vérifier les débuts d'une chaîne

```python
str1 = "Bienvenue"
print(str1.startswith("Bien"))  # Affiche: True
```
