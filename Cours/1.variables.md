#1. Variables

## Qu'est ce qu'une variable ?

Une **variable** est un espace mémoire temporaire qui permet de stocker une valeur sous un nom.
Il en existe plusieurs types (nature de variable) :

| Notation | Types | Définition |
|-----------|-------------|---------|
| **int** | Integer | les nombres entiers (positifs ou négatifs) sans limite de taille. |
| **float** | Float | les Réels, nombres à virgules. |
| **str** | String | les chaine de caractères (= mots, phrase..). |
| **bool** | Booléen | valeurs True (vrai, 1 ) ou False (faux, 0 ). |
| **complex** | Complex | représente les nombres complexe (partie réelle + partie imaginaire). |
| **NoneType** | NoneType | représenté par **None**, utilisé pour indiquer l'absence de valeur. |
| **list** | List | liste ou tableau modifiable. |
| **Tuple** | Tuple | liste ou tableau 'non' modifiable. |
| **dict** | Dictionary | dictionnaire, paires de clé-valeur. |

#### Exemple:

```python
x = 10          # int
pi = 3.14159    # float
nom = "Python"  # str
actif = True    # bool
vide = None     # NoneType
```

---

## Création d'une variable

En Python, on crée une variable simplement en lui assignant une valeur avec le signe `=`.

#### Exemple:
```python
age = 25
prenom = "Alice"
taille = 1.72
user = True
```

---

##  Nommage

Les noms de variables doivent respecter les règles suivantes :

- Commencer par une **lettre** ou un **underscore** `_`
- Contenir uniquement des lettres, chiffres, et underscores
- Ne **pas** commencer par un chiffre
- Ne **pas** utiliser de mots réservés Python (`if`, `for`, `while`, `def`, `class`, etc.)
- Sont **sensibles à la casse** : `age`, `Age` et `AGE` sont trois variables différentes


> [!TIP]
> En Python, on utilise le style `snake_case` pour nommer les variables.


```python
# Noms valides
mon_age = 30
_variable_privee = "secret"
nombre1 = 42

# Noms invalides
# 1age = 10       ← commence par un chiffre
# mon-age = 30    ← tiret interdit
# if = 5          ← mot réservé
```



## 2.7 Conversion de types

On peut convertir une variable d'un type à un autre avec les fonctions de conversion :

| Fonction | Conversion vers |
|----------|----------------|
| `int()` | Entier |
| `float()` | Décimal |
| `str()` | Chaîne de caractères |
| `bool()` | Booléen |

```python
# str → int
age_str = "25"
age_int = int(age_str)
print(age_int + 5)     # 30

# int → float
x = 7
x_float = float(x)
print(x_float)         # 7.0

# int → str
nombre = 42
nombre_str = str(nombre)
print("J'ai " + nombre_str + " ans")   # J'ai 42 ans

# int → bool (0 est False, tout autre entier est True)
print(bool(0))     # False
print(bool(5))     # True
print(bool(-3))    # True
```

> **Attention** : certaines conversions peuvent échouer. Par exemple, `int("bonjour")` lèvera une erreur.

## 2.8 Note sur le vocabulaire et la syntaxe

- **Affecter** / **assigner** : donner une valeur à une variable (`x = 5`)
- **Déclarer** : en Python, toute affectation crée automatiquement la variable
- **Opérande** : les éléments autour d'un opérateur (`a + b` → `a` et `b` sont les opérandes)
- **Expression** : combinaison de valeurs et d'opérateurs (`2 + 3 * x`)
- **Instruction** : une ligne de code complète (`x = 10`)

```python
# Assignation multiple sur une seule ligne 
x, y, z = 1, 2, 3

# Échange de valeurs (swap)
x, y = y, x

# Assignation en chaîne
a = b = c = 0
```

## 2.9 Minimum et maximum

Pour trouver le minimum ou le maximum entre deux valeurs ou dans une collection :

```python
# Entre deux valeurs
a = 10
b = 25
print(min(a, b))   # 10
print(max(a, b))   # 25

# Entre plusieurs valeurs
print(min(3, 7, 1, 9, 4))   # 1
print(max(3, 7, 1, 9, 4))   # 9
```
