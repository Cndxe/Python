#1. Variables

## 2.1 Définition et création

Une **variable** est un espace mémoire qui permet de stocker une valeur sous un nom.

En Python, on crée une variable simplement en lui assignant une valeur avec le signe `=`.

```python
age = 25
prenom = "Alice"
taille = 1.72
user = True
```

---

## 2.2 Les types de variables

| Type | Nom Python | Exemple |
|------|-----------|---------|
| Entier | `int` | `42`, `-7`, `0` |
| Décimal | `float` | `3.14`, `-0.5` |
| Chaîne de caractères | `str` | `"bonjour"`, `'Python'` |
| Booléen | `bool` | `True`, `False` |
| Aucune valeur | `NoneType` | `None` |

```python
x = 10          # int
pi = 3.14159    # float
nom = "Python"  # str
actif = True    # bool
vide = None     # NoneType
```

---

## 2.3 Nommage

Les noms de variables doivent respecter les règles suivantes :

- Commencer par une **lettre** ou un **underscore** `_`
- Contenir uniquement des lettres, chiffres, et underscores
- Ne **pas** commencer par un chiffre
- Ne **pas** utiliser de mots réservés Python (`if`, `for`, `while`, `def`, `class`, etc.)
- Sont **sensibles à la casse** : `age`, `Age` et `AGE` sont trois variables différentes

**Convention** : en Python, on utilise le style `snake_case` pour nommer les variables.

```python
# Noms valides
mon_age = 30
_variable_privee = "secret"
nombre1 = 42

# Noms invalides
# 1age = 10       ← commence par un chiffre
# mon-age = 30    ← tiret interdit
# if = 5          ← mot réservé
```

## 2.4 Écriture scientifique

Pour les très grands ou très petits nombres, Python accepte la notation scientifique avec `e` ou `E` :

```python
grand_nombre = 1.5e6    # 1 500 000.0
petit_nombre = 2.5e-3   # 0.0025
vitesse_lumiere = 3e8   # 300 000 000.0

print(grand_nombre)     # 1500000.0
print(petit_nombre)     # 0.0025
```

## 2.5 Opérations

### Opérations arithmétiques

| Opérateur | Description | Exemple | Résultat |
|-----------|-------------|---------|----------|
| `+` | Addition | `5 + 3` | `8` |
| `-` | Soustraction | `10 - 4` | `6` |
| `*` | Multiplication | `3 * 7` | `21` |
| `/` | Division | `10 / 3` | `3.333...` |
| `//` | Division entière | `10 // 3` | `3` |
| `%` | Modulo (reste) | `10 % 3` | `1` |
| `**` | Puissance | `2 ** 8` | `256` |

```python
a = 10
b = 3

print(a + b)    # 13
print(a - b)    # 7
print(a * b)    # 30
print(a / b)    # 3.3333333333333335
print(a // b)   # 3
print(a % b)    # 1
print(a ** b)   # 1000
```

### Opérations d'assignation combinée

```python
x = 10
x += 5    # équivaut à x = x + 5  → x vaut 15
x -= 3    # équivaut à x = x - 3  → x vaut 12
x *= 2    # équivaut à x = x * 2  → x vaut 24
x //= 5   # équivaut à x = x // 5 → x vaut 4
```

## 2.6 La fonction `type()`

La fonction `type()` retourne le type d'une variable.

```python
x = 42
print(type(x))        # <class 'int'>

y = 3.14
print(type(y))        # <class 'float'>

z = "bonjour"
print(type(z))        # <class 'str'>

b = True
print(type(b))        # <class 'bool'>
```

---

## 2.7 Conversion de types

On peut convertir une variable d'un type à un autre avec les fonctions de conversion :

| Fonction | Conversion vers |
|----------|----------------|
| `int()` | Entier |
| `float()` | Décimal |
| `str()` | Chaîne de caractères |
| `bool()` | Booléen |

```python
# str → int
age_str = "25"
age_int = int(age_str)
print(age_int + 5)     # 30

# int → float
x = 7
x_float = float(x)
print(x_float)         # 7.0

# int → str
nombre = 42
nombre_str = str(nombre)
print("J'ai " + nombre_str + " ans")   # J'ai 42 ans

# int → bool (0 est False, tout autre entier est True)
print(bool(0))     # False
print(bool(5))     # True
print(bool(-3))    # True
```

> **Attention** : certaines conversions peuvent échouer. Par exemple, `int("bonjour")` lèvera une erreur.

## 2.8 Note sur le vocabulaire et la syntaxe

- **Affecter** / **assigner** : donner une valeur à une variable (`x = 5`)
- **Déclarer** : en Python, toute affectation crée automatiquement la variable
- **Opérande** : les éléments autour d'un opérateur (`a + b` → `a` et `b` sont les opérandes)
- **Expression** : combinaison de valeurs et d'opérateurs (`2 + 3 * x`)
- **Instruction** : une ligne de code complète (`x = 10`)

```python
# Assignation multiple sur une seule ligne 
x, y, z = 1, 2, 3

# Échange de valeurs (swap)
x, y = y, x

# Assignation en chaîne
a = b = c = 0
```

## 2.9 Minimum et maximum

Pour trouver le minimum ou le maximum entre deux valeurs ou dans une collection :

```python
# Entre deux valeurs
a = 10
b = 25
print(min(a, b))   # 10
print(max(a, b))   # 25

# Entre plusieurs valeurs
print(min(3, 7, 1, 9, 4))   # 1
print(max(3, 7, 1, 9, 4))   # 9
```
