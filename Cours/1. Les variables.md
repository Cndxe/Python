##  Qu'est ce qu'une variable ?

Les variables stockent temporairement des données en mémoire. 
Il en existe 5 types (nature de variable):

- **int** pour integer --> les nombres entiers (positifs ou négatifs) sans limite de taille.
- **float** --> les Réels, nombres à virgules.
- **str** pour string --> les chaine de caractères (= mots, phrase..).
- **bool** pour boléen --> valeurs True (vrai, 1 ) ou False (faux, 0 ).
- **complex** --> représente les nombres complexe (partie réelle + partie imaginaire).
- **NoneType** --> représenté pas **None** utilisé pour indiquer l'absence de valeur.
